# Claude Control Room

A native macOS desktop app that aggregates all your active Claude CLI sessions into one unified view. See what every session is doing, bring any Terminal window to front, send messages, and get instant summaries — without switching windows.

## Status

Early development. Design doc: `~/.gstack/projects/dex/boluwatifeogunbiyi-main-design-20260410-173325.md`

## Vision

- v1: Live session dashboard (file watcher, observe + navigate)
- v1 MLP: Send messages via clipboard injection, summarize any session on demand
- v2: WebSocket bridge, NLP control, mobile remote
